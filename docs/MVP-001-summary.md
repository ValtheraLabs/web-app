# MVP-001 App Shell

This milestone creates the first runnable Valthera web application shell.

Included:

- Next.js configuration
- TypeScript configuration
- Global CSS visual foundation
- Landing page
- Dashboard page
- Dynamic app section pages
- Shared layout component
- Shared page header component
- Shared feature card component
- Environment example
- Git ignore rules

Routes:

- `/`
- `/app`
- `/app/swap`
- `/app/portfolio`
- `/app/research`
- `/app/ai`
- `/app/settings`

Not included yet:

- Wallet connection
- Blockchain providers
- Backend integration
- AI provider integration
